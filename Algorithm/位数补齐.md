# 位数补齐

## 向上补齐

```c
// 按4补齐
if (stride % 4) {
	stride = (stride & 0xfffc) + 4;
}
```

`0xc`的二进制表示为`0b1100`，如果`stride`非4的倍数，经过按位与运算，最后两位会变为0，即`stride`的值减去了1或2或3，按位与运算的结果是，4的倍数里小于stride且最接近stride的值。

同理可做按2补齐和按8补齐等

```c
// 按2补齐
int d = 19;
if (d % 2) {
    d = (d & 0xfffe) + 2;
}

// 按8补齐
int e = 20;
if (e % 8) {
    e = (e & 0xfff8) + 8;
}
```

**运用按位与运算来进行补齐的方法，只使用与2的指数幂的补齐**
